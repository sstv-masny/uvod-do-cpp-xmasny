
# Task 05 — Sum 1..N

**Input:** Integer `n` with `n >= 0`.  
**Output:** Sum of integers from 1 to `n`. For `n=0`, output `0`.

### Samples
```
in: 0      out: 0
in: 5      out: 15
in: 10     out: 55
```

---

# Úloha 05 — Súčet 1..N

**Vstup:** Celé číslo `n` so `n >= 0`.  
**Výstup:** Súčet celých čísel od 1 do `n`. Pre `n=0` vypíšte `0`.

### Príklady
```
vstup: 0      výstup: 0
vstup: 5      výstup: 15
vstup: 10     výstup: 55
```
